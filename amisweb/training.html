Software Defined Networking

The big idea

Communication network functionality is partitioned into:

Data plane:
the mechanics of transmitting packets through transmission channels and forwarding packets through packet switches
Control plane:
the collection of topology information, and using the topology information along with packet delivery policies (e.g., routing, isolation) to determine how to forward packets through packet switches
SDN is a standard set of abstraction layers for the control plane network functionality. Specifically, abstraction layers for collecting topology information and setting forwarding rules inside packet switches.
The data plane executes in the packet switches.
The control plane executes on computing devices anywhere in the network, this is no longer necessarily distributed throughout the packet switches in the network

Network Measurement

The big idea

Network measurement can provide useful insights for:
  • network engineers to architect network changes to meet requirements
  • security analysts to identify security breaches

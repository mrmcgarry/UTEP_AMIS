Global Science Registry (GSR)

The Global Science Registry is a database created by the GLOBAL RING NETWORK FOR ADVANCED APPLICATIONS DEVELOPMENT (GLORIAD) organization. The GSR contains information that fundamentally permits the mapping of IP addresses and BGP Autonomous System (AS) numbers to Organizational Data (e.g., Organization Name, Address, Geographic coordinates) and vice versa.

As part of the AMIS project we have created a database modeled after the GSR and we hope to merge the two systems in the near future. Access to our system for authorized users can be found here.
